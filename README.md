# Auvexa Dashboard Email Notifications

HTML email templates for Auvexa dashboard transactional emails, configured via Supabase Auth.

## Templates

| File | Description |
|---|---|
| `Invite User.html` | Sent when a user is invited to the Auvexa dashboard |
| `Reset Password.html` | Sent when a user requests a password reset |

## Template Variables

Both templates use Supabase's built-in template variable:

- `{{ .ConfirmationURL }}` — The action link generated by Supabase (invitation or password reset URL)

## Usage

These templates are applied in the Supabase dashboard under:

**Authentication → Email Templates**

Copy the HTML content of each file into the corresponding template editor in Supabase.

## Design

- Brand color: `#0d5463`
- Max email width: `600px`
- Inline styles for maximum email client compatibility
- Table-based layout for broad support across email clients
